# 🎮 astrbot_plugin_80lv | 80.lv 游戏资讯推送

**专业的游戏美术与开发技术资讯推送插件**  
自动抓取 80.lv 最新文章，利用 LLM 进行中文摘要翻译，并生成精美的图文卡片推送到 QQ 群或私聊。

## ✨ 功能特性

- **自动监控**：定时检查 https://80.lv 的最新文章。
- **智能翻译**：调用 AstrBot 配置的 LLM 模型，自动将英文标题和摘要翻译为中文，并保留专业术语。
- **精美排版**：使用 HTML/CSS 渲染生成类似公众号的精美图片卡片，包含封面、作者、标签和摘要。
- **强大的过滤系统**：
  - 支持按 **关键词** 包含/排除。
  - 支持按 **分类标签 (Tags)** 包含/排除。
- **合并转发**：当一次性更新多篇文章时，自动使用合并转发模式，避免刷屏。
