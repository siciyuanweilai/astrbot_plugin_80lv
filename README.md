# 🎮 astrbot_plugin_80lv | 80.lv 游戏资讯推送

**专业的游戏美术与开发技术资讯推送插件**  
自动抓取 80.lv 最新文章，利用 LLM 进行中文摘要翻译，并生成精美的图文卡片推送到 QQ 群或私聊。

## ✨ 功能特性

- **自动监控**：定时检查 https://80.lv 的最新文章。
- **智能翻译**：调用 AstrBot 配置的 LLM 模型，自动将英文标题和摘要翻译为中文，并保留专业术语。
- **精美排版**：使用 HTML/CSS 渲染生成精美的图片卡片，包含封面、作者、标签和摘要。
- **强大的过滤系统**：
  - 支持按 **关键词** 包含/排除。
  - 支持按 **分类标签 (Tags)** 包含/排除。
- **合并转发**：当一次性更新多篇文章时，自动使用合并转发模式，避免刷屏。

## 📥 安装
### 通过 AstrBot 管理面板安装
1. 打开 AstrBot 管理面板。
2. 进入插件市场。
3. 搜索 `astrbot_plugin_80lv` 进行安装。

## ⚙️ 配置
安装后，请在 AstrBot 面板的 插件配置 中进行设置。

### 核心配置项说明
| 配置节点 | 参数项 | 说明 |
| :--- | :--- | :--- |
| **monitor_config** | `enabled` | 是否开启自动监控（建议开启）。 |
| | `interval` | 检查间隔（秒），建议 `300`（5分钟）。 |
| | `receiver` | **重要**：设置接收推送的 `groups` (群号列表) 和 `users` (QQ号列表)。 |
| **filter_config** | `keywords` | **包含关键词**：标题或摘要包含任一词即推送（留空则不限制）。 |
| | `exclude_keywords` | **排除关键词**：包含任一词则**屏蔽**。 |
| | `categories` | **包含分类**：只推送特定标签的文章（如 `Unreal Engine`, `Character Art`）。 |
| | `exclude_categories` | **排除分类**：包含任一标签则**屏蔽**（如 `AI`, `Crypto`, `Sales`）。 |
| **display_config** | `fold` | 合并转发阈值。例如设为 `2`，则更新超过 2 篇时会自动折叠消息。 |

## 💻 指令

| 指令 | 权限 | 说明 |
| :--- | :--- | :--- |
| `/lv check` | 任何人/管理员 | 手动检查一次更新并推送到当前窗口。 |
| `/lv start` | 管理员 | 开启后台自动监控任务。 |
| `/lv stop` | 管理员 | 停止后台自动监控任务。 |

## ⚠️ 注意事项

1. LLM 依赖：本插件依赖 AstrBot 的 LLM 服务进行翻译。。
2. 网络环境：插件需要访问 80.lv，请确保网络可以正常访问该网站。
